    Here’s a list of open source mods you can reference, organized by author. As well, you can also find frameworks at the bottom of this page.

    This is not an extensive list of open source mods available and up-to-date with V Rising. For more check the Thunderstore mod pages for a github link in the top right of their page.

    odjit

    | Mod Name | Description | | — | — | | KindredCommands | Commands to expand Administration efforts and provide information | | KindredSchematics | Commands to build, create, save and load schematics | | KindredPortals | Expand your V Rising fast travel options. Also Map Icons | | KindredArenas | Sets PvP protection buff in designated zones | | KindredLogistics | Logistics system to move items between players and chests |

    CrimsonMods (skytech6)

    | Mod Name | Description | | — | — | | CrimsonFOV | Modifies the client camera field of view | | CrimsonQuest | Questing System | | CrimsonHunt | Create battle-pass like rewards for V Blood hunting | | OfflineRaidGuard | Stops offline raiding, scopes to either clan or player requirements | | CrimsonClans | Extension to Clans: Hearts per Clan and more! | | CrimsonChatFilter | Automated Moderation for Server Chat | | CrimsonBanned | Ban Moderation Tool: Chat, Voice, Server | | CrimsonFAQ | AutoResponse System | | CrimsonDropRate | Modifies the item group drop rate, not the item quantity drop | | AutoBrazier | Automatically turn braziers on/off with day/night cycle | | CrimsonMoon | A sacraficial alter to praise the moon! Honor it in return for a blood moon! | | CrimsonLog | Logging for Moderation |

    Trodi

    | Mod Name | Description | | — | — | | BloodyBoss | BloodyBoss is a mod for V Rising that allows you to create VBlood world bosses with random rewards | | BloodyMerchant | BloodyMerchant is a mod designed for V Rising, offering the capability to create custom in-game merchants | | BloodyMailBox | The BloodyMailBox mod for VRising allows users to receive messages and receive notifications of new messages even when they are offline | | BloodyRewards | Rewards System for VRising | | BloodyShop | Mod to create a store in VRising |

    Rianaid

    | Mod Name | Description | | — | — | | JewelCreator | Server side only mod that allows you to create your own jewels with your chosen skill and skill modifier | | ShardExtraLife | Server side only mod to change shards params like is: durability, time to break, destroy when broken and drop amount | | StarterKit | Server side only mod that allows you to issue different starter kits to new vampires | | MuteChatPlayer | Server side only mod which allows server admins to prohibit a specific player from sending messages to the chat |
